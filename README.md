# Тема лабораторной работы: Создание и базовая настройка приложений Django

## Описание проекта

Этот проект представляет собой веб-приложение, созданное на Django, имитирующее структуру сайта белорусского интернет-магазина настольных игр **znaemigraem.by**.  
Магазин специализируется на продаже настольных игр для детей и взрослых, предлагая как классические игры, так и современные стратегии, головоломки и развлечения для всей семьи.

Основная цель сайта — помочь пользователям разнообразить досуг, организовать весёлый вечер с друзьями или семьёй, а также легко найти и приобрести подходящую игру.

---

## Структура страниц проекта

В проекте реализованы следующие основные страницы:

### 1. **Главная страница** (`base.html`)
- Содержит навигационное меню для перехода на другие разделы сайта.
- Представлена как стартовая точка для пользователей.

### 2. **Страница "О магазине"** (`shop.html`)
- Рассказывает о тематике магазина и его философии.
- Приводит информацию об ассортименте, формате продаж и возможностях сайта (предзаказ, уведомления, личный кабинет).
- Включает примеры игр с ценами, возрастными ограничениями и временем партии.

### 3. **Страница "Об авторе"** (`author.html`)
- Содержит информацию о разработчике лабораторной работы.
- Указаны ФИО, группа и курс автора.

---

## Технические особенности

- Проект построен на Django с использованием шаблонов HTML.
- Каждая страница имеет единый стиль (шрифт Arial, отступы) и навигационные ссылки.
- Реализована возможность возврата на главную страницу с любой другой страницы.
- Дизайн минималистичный, ориентированный на удобство пользователя.

## Целевая аудитория

- Любители настольных игр всех возрастов.
- Семьи, ищущие совместный досуг.
- Компании друзей, организующие игровые вечера.

## Запуск проекта
1. Установите зависимости: `pip install django`
2. Примените миграции: `python manage.py migrate`
3. Запустите сервер: `python manage.py runserver`
4. Откройте в браузере: `http://127.0.0.1:8000/`
